We use JVM languages in most our projects

We often do projects for Estonian customers

Who have special login requirements

e.g. Banklink, ID card, Mobile-ID

So, we decided to make Java developer's life easier

by providing open-source implementations of these login methods

We started with github.com/codeborne/mobileid

(which will eventually become a more general login support Java library)




    // As easy as 1-2-3

    mid = new MobileIDAuthenticator();

    session = mid.startLogin();

    mid.waitForLogin(session);
